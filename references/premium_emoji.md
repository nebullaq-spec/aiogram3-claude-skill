## Использование в сообщениях

```python
# В тексте сообщения
'<tg-emoji emoji-id="5870982283724328568">⚙️</tg-emoji> Настройки'

# В инлайн кнопке
InlineKeyboardButton(
    text="Настройки",
    callback_data="settings",
    icon_custom_emoji_id="5870982283724328568"
)

# В reply кнопке
{"text": "Настройки", "icon_custom_emoji_id": "5870982283724328568"}
```

## Правила

1. В тексте кнопок (`text=`) — **никогда** не ставь обычные эмодзи
2. Всегда используй `icon_custom_emoji_id` для кнопок
3. В тексте сообщений — только `<tg-emoji emoji-id="...">` теги
4. Parse mode должен быть `ParseMode.HTML`
